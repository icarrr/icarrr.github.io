Mudahnya Memasang MSSQL Server di Linux

Microsoft SQL Server di Linux?

Diantara para pembaca mungkin ada yang kaget atau apa kalau produknya Microsoft bisa jalan di linux apalagi tanpa emulator ataupun tools lainnya. Yah itu bisa saja karena mungkin ada yang kemakan omongan bahwa produk Microsoft itu non open source. Dengan adanya MSSQL Server di linux jangan sampai ada terlalu kaget, positif thinking aja mungkin om Bill Gates udah mulai mengopen sourcekan produknya karena kasihan banyak yang berdosa karena menjadi pak tani gelapnya MicrosoftðŸ˜…

Microsoft SQL Server (MSSQL Server) adalah sebuah sistem manajemen basis data relasional produk Microsoft. Bahasa kueri utamanya adalah Transact-SQL yang merupakan implementasi dari SQL standar ANSI/ISO yang digunakan oleh Microsoft dan Sybase. Dimulai dengan SQL Server 2017, SQL Server mulai berjalan di Linux.

Supported Platforms

SQL Server 2017 didukung pada Red Hat Enterprise Linux (RHEL), SUSE Linux Enterprise Server (SLES), dan Ubuntu. MSSQL Server ini juga didukung sebagai Docker image, yang dapat berjalan di Docker untuk Linux atau Docker untuk Windows / Mac.

System Requirements

Untuk menjalankan MSSQL Server di linux, ada persyaratan yang harus dipenuhi yaitu:

Memory 2 GB
File System XFSorEXT4(other file systems, such as BTRFS, are unsupported)
Disk space 6 GB
Processor speed 2 GHz
Processor cores 2 cores
Processor type x64-compatible only

Bermain Dengan MSSQL Server di Linux

  • Installasi di CentOS dan saudara sepaket manajernya

    1) Tambahkan repositori MSSQL Server ke sistem

    sudo curl -o /etc/yum.repos.d/mssql-server.repo https://packages.microsoft.com/config/rhel/7/mssql-server-2017.repo

    2) Install MSSQL Server

    sudo yum install -y mssql-server

    3) Setup MSSQL Server, Setup disini akan membuat password untuk akun SA karena secara default MSSQL Server menggunakan user SA

    sudo /opt/mssql/bin/mssql-conf setup

  • Installasi di Ubuntu dan saudara sepaket manajernya

    1) Tambahkan GPG key MSSQL Server ke sistem

    wget -qO- https://packages.microsoft.com/keys/microsoft.asc | sudo apt-key add -

    2) Tambahkan repositori MSSQL Server ke sistem

    sudo add-apt-repository "$(wget -qO- https://packages.microsoft.com/config/ubuntu/16.04/mssql-server-2017.list)"

    3) Install MSSQL Server

    sudo apt-get update && sudo apt-get install -y mssql-server

    4) Setup MSSQL Server, Setup disini akan membuat password untuk akun SA karena secara default MSSQL Server menggunakan user SA

    sudo /opt/mssql/bin/mssql-conf setup
