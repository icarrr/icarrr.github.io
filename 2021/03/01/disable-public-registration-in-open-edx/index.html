Disable Public Registration in Open edX

To disable public registration in Open edX, follow the steps below:

  1. Go to Open edX server console.

  2. Open and edit LMS and CMS configuration (lms.env.json and cms.env.json).

  3. Then find the "FEATURES" configuration, add ALLOW_PUBLIC_ACCOUNT_CREATION into "FEATURES" array and set to false. See the following codes snippet:

    "FEATURES": {
        ...
        "ALLOW_PUBLIC_ACCOUNT_CREATION": false,
        ...
    },
  1. Restart LMS services.
