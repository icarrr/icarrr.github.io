Integrate Zoom With Open edX

This topic contains how to setup Zoom LTI Pro, enable LTI Consumer and add the LTI component to the course.

Setup Zoom LTI Pro

  1. Go to marketplace zoom.

  2. Select LTI Pro.

  1. Enable “Pre-approve” and click “Install”.

  1. LTI Pro will be request access to your zoom account, select “authorize”.

  1. Create a new credential.

  1. Choose your credential name, and then click “Save”.

  1. On LTI Credentials page, your need edit/update this:
  • on field “Email Attribute Name”, add instructor_email
  • on field “Approved Domains”, add your LMS and CMS url
  1. Copy your LTI Key and LTI Secret.

Until this stage, LTI Pro has been set up.

Enable LTI Consumer on Open edX

The XBlock implements the consumer side of the LTI specification which supports the integration of third party LTI provider tools.

For add component LTI on your course, LTI Consumer need to enable.

  1. From studio course page, navigate to Settings -> Advanced Settings.

  1. Find the key Advanced Module List and add “lti_consumer” to Advanced Module List. Etc:
[
    "lti_consumer"
]
  1. Find the key LTI Passports and add LTI passports using format "zoom:client_key:client_secret". Etc:
["zoom:P0f6ykC78EYnwRQG:zpvyKVXvwhYthjMj1ighKtssJoONglNY"]
  1. Click Save changes.

Add the LTI Consumer to the course

  1. Click component Advanced and select LTI Consumer.

  1. Once a Consumer LTI component is selected, edit it.
  • LTI Application Information: This field is a description of the third party application
  • LTI ID: add the id entered in the previous step, namely in LTI Passports
  • LTI URL: https://applications.zoom.us/lti/rich
  • Custom Parameters: [“instructor_email=yourzoomemail@mail.com”]
  1. Click Save.
