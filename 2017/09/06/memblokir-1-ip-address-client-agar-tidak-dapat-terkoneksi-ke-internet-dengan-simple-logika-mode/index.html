Memblokir 1 IP Address Client Agar Tidak Dapat Terkoneksi ke Internet Dengan Simple Logika Mode

A. Pendahuluan

Pada praktek ini, saya akan memposting bagaimana memblokir ip address dari client agar tidak dapat terhubung ke internet tetapi masing menggunakan firewall simple logika

B. Pengertian

Firewall simple logis/logika adalah konfigurasi firewall secara logis sehingga bisa lebih mudah dipahami karena lebih manusiawi

C. Latar belakang

Ingin menambah ilmu kemudian untuk disharing

D. Tujuan

Agar client tidak bisa terhubung ke internet apabila ia mendapatkan ip yang telah diblokir

E. Hasil yang diharapkan

Dapat melakukan konfigurasi dengan cepat, baik, dan benar

F. Alat dan bahan

  1. Mikrotik
  2. Koneksi internet
  3. Pc/laptop

G. Jangka waktu pelaksanaan

Waktu yang dibutuhkan sekitar 5-10 menit

H. Langkah kerja

  1. Pada praktek ini saya akan memblokir ip sendiri, namun sebelumnya dipastikan terlebih dahulu bahwa saya dapat mengakses internet

  1. Hubungkan mikroTik dengan perangkat > buka winbox > login untuk konfigurasi mikroTik

  2. Cek ip yang konek di mikroTik kita sendiri : IP > ARP. Lalu ingat ip yang akan kita cegah untuk akses internet atau bisa juga dengan doube klik pada ip yang akan dicegah > block pada ip > copy

  1. Atur firewall untuk memblokir ip : IP > Firewall > Filter rules > add (+)

tab general :

  • chain = forward
  • src.address = ip yang akan dicegah untuk internet, misal ip saya 192.168.1.6
  • out.interfaces = sumber internet, misal eth1

tab action :

  • action = drop

  1. Karena saya memblok ip sendiri, maka dicoba untuk mereload situs yang terbuka sebelumnya dan sekarang tidak bisa, hanya connecting terus

I. Hasil yang didapatkan

Dapat melakukan konfigurasi dengan baik dan benar sehingga munculnya permasalahan dapat diminimalisir

J. Temuan masalah

Tidak ditemukan permasalahan untuk praktek kali ini

K. Kesimpulan

Apabila ada client yang tiba-tiba terhubung ke jaringan kita namun secara kebetulan ia mendapatkan ip yang telah diblokir maka ia tidak dapat tehubung ke internet
