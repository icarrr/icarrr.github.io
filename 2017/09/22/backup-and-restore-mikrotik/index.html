Backup and Restore Mikrotik

1. Pendahuluan

A. Pengertian

Di bidang teknologi informasi, backup, atau proses back up, mengacu pada penyalinan dan pengarsipan data sehingga bisa digunakan untuk mengembalikan dokumen asli setelah terjadi peristiwa kehilangan data. Bentuk kata kerja adalah untuk membuat cadangan dua kata, sedangkan kata benda adalah cadangan.

Restore , mengacu pada proses pengembalian data menggunakan file backup.

B. Latar Belakang

Membackup data mikrotik agar saat perangkat rusak kita punya salinan konfigurasinya.

C. Hasil Yang Diharapkan

Dapat melakukan Backup dan restore untuk menyalin data konfigurasi, dan mengembalikannya jika perangkat ganti (versi perangkat sama dengan yang lama).

2. Alat dan Bahan

  • Router Mikrotik
  • Kabel Lan
  • Laptop

3. jangka Waktu Pelaksanaan : 30 Menit

4. Langkah - Langkah Pelaksanaan

A. buka winbox , untuk konfigurasi mikrotik

B. Masuk ke menu file

C. Lalu klik backup

D. Isikan nama untuk file backup dan password (untuk password bisa dikosongkan) , tunggu hingga backup selesai. File Backup Bisa di download melalui FTP.

E. Untuk Restore sama masuk kemenu file lalu pilih fileyang akan direstores lalu tunggu hingga restore selesai.

5. Hasil Yang Didapat

Berhasil Melakukan backup dan restore

6. Kesimpulan

Backup dan restore pada mikrotik dapat kita manfaatkan untuk menyalin data konfigurasi agar saat perangkat rusak kita punya masi punya konfigurasinya , dan kita dapat merestore nya ke perangkat baru . dengan catatan perangkatr dengan versi yang sama.
