Pointing UBNT ke MikroTik

PENGERTIAN

Point to Point adalah data link protokol yang umum digunakan dalam membangun hubungan langsung antara dua node jaringan. Hal ini dapat menyediakan koneksi otentikasi, transmisi enkripsi (menggunakan ECP, RFC 1968), dan kompresi.

LATAR BELAKANG

Jauhnya tempat tidak memunginkan untuk penggunakan kabel untuk media transmisi data

MAKSUD DAN TUJUAN

Mempelajari UBNT

Jangka Waktu Pelaksanaan

Kurang lebih 3 jam

ALAT DAN BAHAN

~ Laptop

~ Ubnt Powerbeam M2

~ RB 941

~ Alat untuk memnjat tower, Tali Wabbing, Harnesses, Tas untuk wadah kunci,

~ Kabel LAN

TAHAPAN

Seperti itu topologi dari Pointing ini,

pertama konfigurasi Mikrotik sebagai AP Bridge

disini saya menggunakan Routing Static jadi harus memasukkan DNS sendiri, dan dibawah adalah gambar konfigurasi Ubnt, Default untuk Ubnt adalah 192.168.1.20, satu subnetkan IP Laptop NOC agar bisa mengakses web Ubnt tersebut.

default user dan password adalah

username: ubnt
password: ubnt

setelah itu kita atur Sistemnya, disini saya hanya mengganti passwordnya saja

lalu kita masuk tab wireless dan kita scan, seperti gambar diatas. Setelah SSID AP ketemu langsung ita lock saja

setelah di lock kita samakan frequensi dengan frequensi mikrotik tenda, lalu kita masukkan wireless security bila menggunakan password.

pada menu Network, kita konfigurasi IP agar satu subnet.

nah sekarang pada menu Main kita bisa memantau seperti gambar diatas. disini kita akan melakukan Pointing dengan menyamakan TX/Rx rate dan sinyal strenght nah disini harus ekstra sabar baik si pemanjat tower dan yang memantau dibawah, agar mendapat hasil yang bagus pada sinyal yang terbaik adalah yang memiliki dBm sedikit.

sedangkan TX/Rx nya harus 1:1.

Kesimpulan

Untuk Pointing dengan Ubnt harus di tangani oleh ahlinya karena saat mencari TX/RX dan Sinyal sedikit susah karena berada di ketinggian tertentu.
