Hal-Hal Yang Harus Dilakukan Setelah Install Server - Install Aplikasi Berbasis Web

A. Pendahuluan

Pada kesempatan ini, saya kembali memposting tentang dasar-dasar setelahi install server.

B. Pengertian

Aplikasi Berbasis Web adalah sebuah aplikasi yang dapat diakses melalui internet atau intranet, dan pada sekarang ini ternyata lebih banyak dan lebih luas dalam pemakaiannya.

C. Latar belakang

Setelah kita menginstall debian di proxmox tentunya kita harus tau hal hal apa saja yang harus dilakukan setelahnya.

D. Tujuan

Agar server dapat bekerja secara optimal saat melayani komputer client.

E. Hasil yang diharapkan

Dapat menginstall aplikasi web server yang lain

F. Alat dan bahan

  1. Laptop/pc
  2. Aplikasi web yang akan diinstall
  3. Debian server

G. Jangka waktu pelaksanaan

Waktu yang dibutuhkan kisaran 5-10 menit

H. Langkah kerja

1. Aktifkan module yang diperlukan, misal seperti berikut

2. Untuk mengaktifkan konfigurasi apache2 meminta agar servicenya direstart

3. Install PHP extention / paket lain yg diperlukan, misal seperti berikut

4. Copy file aplikasi yang ingin diinstall ke directory server

5. Install paket unzip agar dapat mengestrak file yang telah dicopy

6. Ekstrak file yang telah dicopy

7. Berikan hak akses sebagai root untuk menjalankan aplikasi

8. Buat database melalui phpmyadmin, import database apabila diperlukan

I. Hasil yang didapatkan

Dapat menginstall aplikasi berbasis web

J. Temuan masalah

Belum ditemukan masalah

K. Kesimpulan

Dengan aplikasi berbasis web, client tidak perlu menginstall aplikasi itu juga melainkan bisa mengakses melalui web browser.
