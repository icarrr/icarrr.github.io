Setting TP-LINK WA7210N


A.Pendahuluan

Assalamualaikum Wr.Wb
saya akan membuat tutorial yang berjudul cara setting tp link agar bisa share wifi nya . jika kalian ingin tau caranya saksikan tutorialini dengan baik jagan ada kelewatan.

B. Pengertian 

TP-LINK itu jenis nya berbagai macam tapi yang saya akan buat tutorial ini yaitu cara setting tplink. menurut saya TP-LINK adalah sebuah alat untuk memperkuat sinyal maupun membuat hospot dengan berskala besar.

C. latar belakang

Diagram contoh ini digunakan untuk menghubungkan Wireless Router TP-Link TD-W8951ND dengan koneksi internet ADSL dan wireless WIFI, signal WIFI dari Router Modem ADSL akan diambil TP-Link TL-WA7210N. Dari TL-WA7210N kembali disebarkan menjadi signal WIFI yang dapat diterima oleh smartphone, tablet atau notebook dengan wireless.

D. Maksud dan tujuan 

maksud dari tutorial berikut adalah untuk mengetahui cara membuat hospot dari tp link tersebut dan kalian juga dapat membuat jaringan point to point dengan komputer server

E. Alat dan bahan

  1. TP-LINK WA7120N
  2. komputer
  3. jaringan internet yang mau di pakai untuk membuat hospot
  4. kabel LAN

F. langka kerja

1.langkah pertama masukkan ip address TP-Link kalian ke web browser jika Tp-link kalian sama dengan yang saya pakai itu ip nya dalah 192.168.1.254 jika kalian sudah masuk ke ip nya maka akan muncul perintah untuk mamasukkan user name dan pasword jika kalian tidah tahu maka kalian masukkan password dan user name admin dan adminseperti gambar berikut

2. jika kalian sudah masukkan password dan username maka kalian langsung next aja.

3. katika kalian sudah masuk ke tp link kalian. jika kalian ingin membuat hospot maka kalian masuk ke quick setup seperti di bawah ini.

4. jika kalian sudah masuk ke quick setup kalian tingal pilih aja saya disini menggunakan akses point seperti gambar di bawah


5. jika kalian sudah masuk ke ap maka kalian masukkan konfigurasi tp link kalian sesuai gambar berikut.

6.kemudian kalian tinggal finish aja. jika kalian mau masukkan ssid nya  makakalian ikuti cara berikut.
