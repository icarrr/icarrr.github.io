Mempercantik tampilan terminal server

A. Pendahuluan
Pada kesempatan ini saya ingin sharing tentang bagaimana mempercantik atau apalah agar tampilan shell linux tidak polos-polos begitu saja.

B. Pengertian
Apa itu shell ? shell adalah program (penterjemah perintah) yang menjembatani user dengan sistem operasi dalam hal ini kernel (inti sistem operasi), umumnya shell menyediakan prompt sebagai user interface, tempat dimana user mengetikkan perintah-perintah yang diinginkan baik berupa perintah internal shell (internal command), ataupun perintah eksekusi suatu file progam (eksternal command), selain itu shell memungkinkan user menyusun sekumpulan perintah pada sebuah atau beberapa file untuk dieksekusi sebagai program.

C. Latar belakang
Bosan melihat tampilan shell yang defaut atau yang masihp polos-polos

D. Tujuan
Agar lebih semangat lagi ngoprek menggukan shell

E. Hasil yang diharapkan
Dapat mengetahui bagaimana prosesnya bukan hanya hasilnya saja

F. Alat dan bahan
  1. Shell linux
  2. Laptop/pc
  3. Akses internet
G. Jangka waktu pelaksanaan
Waktu yang dibutuhkan kisaran 5-10 menit

H. Langkah kerja
1. Remote server anda, kemudian masuk sebagai super user/root

2. Install figlet dengan perintah seperti berikut

3. Edit file .bashrc dengan perintah sebagai berikut

4. Tambahkan script seperti berikut dibaris paling bawah/akhir

5. Keluar dari remote server, kemudian remote lagi server anda dan tampilan shell nya sudah ada perubahan


6. Untuk menambahkan selain dari figlet, anda harus menginstall paketnya terlebih dahulu

7. Edit lagi file .bashrc kemudian tambahkan script berikut


8. Keluar dari remote server, kemudian remote lagi server anda dan tampilan shell nya sudah ada perubahan

9. Untuk cowsay yang lain, anda dapat melihat varian yang disediakan yaitu dengan perintah berikut

10. Kemudian anda ganti di .bashrc yang saya beri tanda merah

I. Hasil yang didapatkan
Dapat merubah tampilan awal shell linux

J. Temuan masalah
Di praktek ini saya mengalami masalah karena, kesalahan memasukkan perintah di .bashrc

K. Kesimpulan
Tampilan shell linux dapat diubah agar brainwarenya tidak bosan
