Tips Menemukan Referensi Website Untuk Belajar Linux

A. Pendahuluan

Halo teman2 blogger, pada kesempatan kali ini saya akan mmebagikan sebuah artikel sederhana namun bermanfaat bagi pembaca sekalian terutama bagi yang ingin belajar linux agar mahir menggunakannya. Monggo mari di simak artikel yang saya bagikan berikut ini.

1. Pengertian

Web referensi adalah suatu website atau situs atau laman yang memuat pembahasan ataupun tutorial yang berisi materi yang dapat di jadikan patokan atau dasar untuk membuat suatu artikel ataupun tutorial.

2. Latar Belakang

Kita sering mendapati web referensi yang kurang relevan karena berisi materi atau artikel yang kurang bisa di jadikan sebagai patokan atau dasar sehingga kita perlu menemukan web referensi yang memuat artikel yang sudah “terpercaya” atau bisa di bilang “resmi”.

3. Maksud dan Tujuan

Mengetahui apa itu web referensi dan mengenalkan macam2 web referensi yang berisi artikel atau konten yang relevan.

4. Hasil yang Di Harapkan

Pembaca dapat menemukan web referensi artikel atau konten yang relevan.

  • Koneksi Internet

D. Proses dan Tahapan Pelaksanaan Kegiatan

Berikut TIPS cara mencari referensi website untuk mempermudah belajar linux sobat, ini hanyalah pendapat pribadi penulis (bukan keharusan) :

  • Gunakan contekan COMMAND LINE (perintah TERMINAL) kalau sudah hafal tidak usah tidak apa2 : http://www.linuxguide.it/command_line/linux_commands_in.html
  • Tutorial Vi/Vim Editor http://www.yolinux.com/TUTORIALS/LinuxTutorialAdvanced_vi.html
  • Tutorial IPTables : http://rootbox.or.id/tips/iptables.html
  • How To Forge : http://www.howtoforge.com/
  • Server World : http://www.server-world.info/en
  • YoLinux : http://www.yolinux.com/
  • nixCraft : http://www.cyberciti.biz/
  • Unixmen : http://www.unixmen.com/
  • Ubuntu Server Guide : http://ubuntuserverguide.com/
  • Wiki Onno W. Purbo : http://opensource.telkomspeedy.com/wiki/index.php/Main_Page

E. Hasil yang Di Dapatkan

Dapat menemukan web referensi yag terpercaya dan mengandung konten yang relevan.

F. Temuan Permasalahan

Belum menemukan masalah.

G. Kesimpulan

Dapat di simpulkan bahwa website yang relevan akan mempengaruhi kelancaran pembuatan artikel atau tutorial yang akan di buat.
