Installasi MATE Desktop Environment di Ubuntu 17.04

A. Pendahuluan

Setelah kemarin menggunakan desktop gnome saya ingin pindah ke desktop MATE karena lebih ringan dan mudah digunakan, nah bagi kalian yang ingin menginstall MATE desktop kali ini saya bagikan tutorialnya…

B. pengertian

MATE telah memiliki sejumlah aplikasi yang berasal dari Aplikasi Inti GNOME, dan pengembang telah menulis beberapa aplikasi lain dari nol. Aplikasi bercabang memiliki nama baru, kebanyakan dari bahasa Spanyol.

MATE juga tersedia di repositori resmi beberapa distribusi Linux lainnya, termasuk Arch Linux, Parabola, LMDE, Debian, Ubuntu MATE, Ubuntu Mint, Kali Linux, Fedora, openSUSE, Gentoo, Mageia, ROSA, dan PCLinuxOS. Selain itu, ada repositori pihak ketiga untuk Slackware. Versi 3.5 dan versi GhostBSD termasuk MATE sebagai lingkungan desktop default, menjadikannya sebagai masuknya MATE sebagai desktop default, setelah Linux Mint, dan yang pertama di OS non-Linux. Ini juga tersedia untuk FreeBSD (bersama turunannya, seperti PC-BSD) dan NetBSD dan default pada OpenIndiana.

F. alat dan bahan

  • laptop
  • OS ubuntu
  • koneksi internet

G. jangka waktu

5 menit

H. tahap pelaksanaan

  1. hidupkan laptop dan boot ke ubuntu kalian
  2. buka terminal ctrl+T
  3. ketikan perintah

sudo apt get update

  1. lanjut dengan installasi MATE

sudo apt-get install mate-desktop-environment-core

  1. lalu install desktop environmentya

sudo apt-get install mate-desktop-environment

  1. install packetnya juga

sudo apt-get install mate-desktop-environment-extras

  1. jika sudah, kalian log out user kalian dan ubah desktopnya di pojok kanan atas dan login kembali

I. hasil yang didapatkan

Penggunaan MATE menurut saya ringan dan mudah di install

J. permasalahan

Desktop MATE ini sebenarnya seperti desktop yang dimiliki oleh linux mint MATE, parrot dan lainya, jadi kalau agak mirip dengan yang lain jangan kaget

K. kesimpulan
