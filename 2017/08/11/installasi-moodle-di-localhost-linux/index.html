Installasi Moodle di Localhost Linux


A. Pendahuluan 
Server dibuat dengan beragam maksud dan tujuannya, salah satunya adalah membangun sistem pembelajaran online. Pembelajaran online sangat baik diterapkan saat ini, karena pelajar masa kini lebih senang untuk online dari pada membaca buku. Hal ini bisa meningkatkan minat membaca pelajar tanah air. Salah satu media yang dapat digunakan adalah dengan “Moodle’. Apa itu Moodle? 

#Pengertian
Moodle (Modular Object-Oriented Dynamic Learning Environment) adalah paket perangkat lunak yang diproduksi untuk kegiatan belajar berbasis internet dan situs web yang menggunakan prinsip social constructionist pedagogy. Moodle merupakan salah satu aplikasi dari konsep dan mekanisme belajar mengajar yang memanfaatkan teknik informasi, yang dikenal dengan konsep pembelajaran elektronik atau e-learning. Moodle dapat digunakan secara bebas sebagai produk sumber terbuka (open source) dibawah lisensi GNU. Moodle dapat diinstal dikomputer dan sistem operasi apapun yang bisa menjalankan PHP dan mendukung database SQL.
Sistem pembelajaran elektronik atau e-learning dapat didefinisikan sebagai sebuah bentuk teknologi informasi yang diterapkan di bidang pendidikan berupa website yang dapat diakses dimana saja. E-learning merupakan dasar dan konsekuensi logis dari perkembangan teknologi informasi dan komunikasi. Dengan e-learning, murid tidak perlu duduk dengan manis diruang kelas untuk menyimak setiap ucapan dari seorang guru secara langsung. E-learning juga dapat mempersingkat jadwal target waktu pembelajaran, dan tentu saja menghemat biaya yang harus dikeluarkan oleh sebuah program studi atau program pendidikan. 

#Latar Belakang
Pelajar saat ini lebih senang membaca dimedia internet ketimbang buku biasa. 

#Maksud dan Tujuan
Mengetahui apa itu Moodle dan menerapkannya dalam sistem pembelajaran 
#Hasil Yang Diharapkan  
Mampu mengkonfigurasikan Moodle dengan benar sehingga kita dapat menerapkan sistem pembelajaran online demi upaya meningkatkan minat baca peserta.

B. Alat dan Bahan
1. Komputer Server
2. Laptop
3. Akses Internet
4. Download file Moodle disini 

C. Jangka Waktu Pelaksanaan
Jangka waktu pelaksanaan 2 jam

D. Tahapan Pelaksanaan
1. Download file moodle dari situs resminya,kemudian dari directory download tersebut klik kanan lalu pilih open in terminal.

2. Copy file moodle yang telah didownload ke directory /var/www/html

3. Ketik “cd” pada terminal agar kembali ke directory awal lalu masuk ke directory /var/www/html

4. unzip file moodle yang telah di copy

5. Masuk ke mode super user lalu berikan hak akses


6. Kembali ke directory /www, lalu buat directory moodledata


7. Berikan hak akses untuk directory tersebut

8. Buat database di phpmyadmin

9. Install moodle dengan membuka moodle melalui browser caranya ketik localhost/moodle, lalu pilih next

10. Next terus hingga di pemilihan driver database, untuk typenya pilih MariaDB > next

11. Isi database host, database name, database user, dan database password. Untuk database user dan password, isi sesuai user dan password phpmyadmin > next

12. Setelah itu klik Continue

13. Ketika muncul masalah seperti berikut, klik salah satu peringatan

14. Akan muncul perintah untuk mengkonfigurasi file my.cnf yang ada di directory mysql, copy script tambahan yang ditampilkan

15. Pindah ke directory mysql

16. Ketikkan perintah untuk mengedit file my.cnf

17. Scroll ke script paling akhir > enter, lalu paste script yang telah dicopy


18. restart mysql

19. Buka browser reload page installation moodle, dan hasilnya sudah berbeda. klik Continue dan tunggu beberapa saat

20. Masukkan data-data dengan benar lalu klik continue

21. Masukkan nama lengkap situs dan nama pendek situs lalu continue

22. Moodle telah berhasil diinstal


E. Hasil Yang Didapatkan
Dapat menginstall moodle dengan benar
