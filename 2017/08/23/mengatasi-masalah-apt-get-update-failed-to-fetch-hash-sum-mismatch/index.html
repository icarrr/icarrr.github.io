Mengatasi Masalah 'apt-get update failed to fetch hash sum mismatch'

A. Pendahuluan

Pada kali ini, saya akan memposting tentang bagaimana mengatasi masalah yang biasa ditemukan saat melakukan update linux.

B. Pengertian

Shell adalah program (penterjemah perintah)yang menjembatani user dengan sistem operasi dalam hal ini Kernel (inti dari sistem operasi), umumnya shell menyediakan prompt sebagai user interface, tempat dimana user dapat mengetikkan perintah-perintah yang diinginkan baik berupa perintah internal shell (internal command), ataupun perintah eksekusi file program (eksternal command), selain itu memungkinkan user menyusun sekumpulan perintah pada sebuah atau beberapa file untuk dieksekusi sebagai program. Salah satu perintah dari shell yaitu apt-get update, namun terkadang mengalami error saat menjalankan perintah tersebut.

C. Latar belakang

Beberapa hari yang lalu, saya pernah mengalami masalah saat mengeksekusi perintah apt-get update di shell linux.

D. Tujuan

Membantu orang-orang untuk mengatasi masalah yang serupa.

E. Alat dan bahan

  1. Pc/laptop

F. Jangka waktu pelaksanaan

Waktu yang dibutuhkan untuk mengatasi masalah tersebut kurang lebih 1 menit saja.

G. Langkah kerja

  1. Buka terminal/shell linux kemudian masuk sebagai root

  2. Ketik perintah berikut untuk menghapus beberapa file yang merusak proses update

> rm -rf /var/lib/apt/lists/*
  1. Coba ketik perintah apt-get update dan jrengg jrengg errornya sudah hilang

H. Temuan masalah

Tidak ada masalah yang ditemukan

I. Hasil yang didapatkan

Dapat mengatasi masalah failed to fetch hash sum mismatch dengan baik dan benar.

J. Kesimpulan

Masalah tersebut disebabkan karena ada beberapa file yang merusak file repo yang tersimpan dilinux, makanya untuk mengatasinya caranya yaitu mengahapus file tersebut dengan perintah yang telah diberikan di post ini.
