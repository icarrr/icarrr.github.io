Bagaimana Menginstall LAMP di Localhost Linux?

A. Pendahuluan

Hello, pada kesempatan kali ini saya memposting bagaimana sih menginstal LAMP? Nah cara-caranya akan dijelaskan sebagai berikut.

#Pengertian

LAMP adalah istilah yang merupakan singkatan dari Linux, Apache, MySQL dan Perl/PHP/Phyton. Merupakan sebuah paket perangkat lunak bebas yang digunakan untuk menjalankan sebuah aplikasi secara lengkap.

Komponen-komponen dari LAMP :

  • Linux – sistem operasi
  • Apache HTTP Server – web server
  • MariaDB atau MySQL – sistem basis data
  • PHP atau Perl atau Python – bahasa pemrograman yang dipakai

#Latar Belakang

Ingin menggunakan linux, apache, mariaDB/mysql, dan PHP dalam satu paket.

#Maksud dan Tujuan

Maksud dan tujuan dari postingan ini agar si pengunjung dapat mempelajari dan mengetahui bagaimana menginstall LAMP.

B. Alat dan Bahan

  1. Komputer/laptop
  2. Koneksi internet

C. Jangka Waktu Pelaksanaan

Waktu yang diperlukan kurang lebih 20-30 menit

D.** Langkah Kerja**

1. Buka terminal liinux, dan masuk sebagai super user (su)

2. Install apache

3. Install php dan beberapa dependensinya

4. Buat sebuah file “testphp.php” > “nano testphp.php” dan sisipkan skrip berikut

5. Cek browser anda dan ketik “localhost/testphp.php” pada address bar, jika tampilannya tidak menampilkan versi php anda silahkan install yang ini. Jika iya, abaikan bagian ini

6. Install mariadb-server

7. Konfigurasi mysql

  • Tekan enter

  • Masukkan password

  • Remove user anonymous

  • Matikan remote login

  • Remove database dan reload table

8. Install phpmyadmin

9. Pada saat menginstall phpmyadmin, akan di tampilkan sebuah jendela seperti berikut. Tanda i apache dengan cara klik spesialis > ok

10. Pilih yes

11. Masukkan password, lalu konfirmasi pass tersebtut

12. Cek browser anda, ketik “localhost/phpmyadmin” pada address bar

13. Silahkan login dengan username “phpmyadmin” dan password “sesuai yang telah di set sebelumnya”.

14. PHPMYADMIN anda sudah selesai.

E. Hasil Yang Didapatkan

Mampu mempraktekkannya dengan sendiri dan benar

F. Kesimpulan

LAMP itu simliar ataupun samadengan XAMP yang mempuyai apache, mysql.
