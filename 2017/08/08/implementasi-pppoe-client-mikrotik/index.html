Implementasi PPPoE client Mikrotik

A. PENGERTIAN
PPPOE merupakan singkatan dari point-to-point protocol over Ethernet yang merupakan protokol dalam jaringan untuk menghubungkan komunikasi antara dua buah titik jaringan atau dua buah port ethernet dengan model tunneling (terowongan) dan juga sebagai virtual dial-up dalam jaringan. mikrotik adalah salah satu router yang mendukung konfigurasi pppoe server dan client, untuk itulah akan di tuliskan pada artikel ini tentang bagaimana membuat pppoe server dengan mikrotik.

B. LATAR BELAKANG

4. Kemudian ke PPPOE client

dan pppoe client dah jadi

6. kemudian ke ip > firewall

7. kemudian ke NAT

8. dalam NAT silahkan ke menu General > dan isi
chain = srcnat
out-interface = pppoe1-out
kemudian ke action dan isi kolom action dengan > masquerade > kemudian apply dan ok
