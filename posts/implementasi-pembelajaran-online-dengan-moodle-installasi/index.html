Implementasi pembelajaran online dengan moodle - installasi
A. Pendahuluan
Pada kesempatan kali ini saya akan mengimplementasikan tentang apa yang sudah saya pelajari yaitu Moodle

B. Pengertian
MOODLE (singkatan dari Modular Object-Oriented Dynamic Learning Environment) adalah paket perangkat lunak yang diproduksi untuk kegiatan belajar berbasis internet dan situs web yang menggunakan prinsip social constructionist pedagogy. MOODLE merupakan salah satu aplikasi dari konsep dan mekanisme belajar mengajar yang memanfaatkan teknologi informasi, yang dikenal dengan konsep pembelajaran elktronik atau e-learning. Moodle dapat digunakan secara bebas sebagai produk sumber terbuka (open source) di bawah lisensi GNU. Moodle dapat diinstal di komputer dan sistem operasi apapun yang bisa menjalankan PHP dan mendukung database SQL.

C. Latar belakang
Melakukan pembelajaran online bagi siswa disekolah maupun umum.

D. Tujuan
Agar pembelajaran secara online dapat menarik siswa atau kalangan umum untuk ikut belajar.

E. Hasil yang diharapkan
  1. Dapat mengimplementasikan Moodle dengan baik.
  2. Pembaca memahami dan dapat mempraktekanyya

F. Alat dan bahan
  1. file master moodle
  2. server
  3. pc/laptop
G. Jangka waktu pelaksanaan
Waktu yang dibutuhkan untuk membuat atau mengaktifkan Moodle sekitar 20 menit.

H. Langkah kerja
1. Copy file master moodle ke server

2. Remote server dan cek file yang dicopy tadi, jika ada pindahkan ke folder /var/www/html/


3. Jika proses copy sudah selesai kemudian kita masuk ke folder html

4. Kemudian jika sudah kita extrax file moodlenya dengan perintah seperti pada gambar dibawah ini.

5. Jika sudah diextrax maka kita ls akan muncul nama moodle seperti gambar dibawah ini.

6. Kita rubah hak akses untuk moodle dengan perintah seperti pada gambar dibawah ini.

7. Kemudian kita masuk ke folder /var/www/

8. Kita buat folder baru didalam /var/www/ untuk moodledata/, kemudian kita cek menggunakan ls maka akan muncul folder moodledata.

9. Kemudian kita buat hak akses untuk moodledata menggunakan perintah seperti pada gambar dibawah ini.

10. Setelah itu kita buat Database untuk moodle sendiri di phpmyadmin

11. setelah itu kita bisa buka ke browser dengan mengetikan ip/moodle maka akan muncul tampilan seperti pada gamabr dibawah ini kita pilih english untuk language

12. Kemudian kita pilih MariaDB pada type database

13. Kemudian isi seperti pada gambar dibawah ini pada database password isikan password phpmyadmin

14. Jika muncul error seperti pada gambar dibawah ini maka kita klik yang sudah saya garisin merah

15. Maka akan muncul tampilan seperti gambar dibawah ini kemudian kita copy script yang ada pada halaman tersebut.

16. Kemudian kita ketikkan perintah seperti gambar dibawah ini

17. Setelah itu kita pastekan script yang tadi sudah dicopy, tekan ctrl+x dan Y untuk keluar dan menyimpan

18. Setelah itu kita restart mysql kita

19. kemudian kita buka kembali browser lalu reload page yang tadi maka akan muncul tampilan seperti dibawah ini kemudian kita cari kebawah klik continue


20. Setelah itu kita isikan data dengan benar untuk moodle yang kita buat

21. Setelah selasai mengisi data dengan benar kita klik save change

22. Kemudian isikan Full site name dan Short name for site isikan bebas terserah anda

23. Jika berhasil maka tampilannya seperti pada gambar dibawah ini.
