Enable Timed Exams on Open edX

Timed exams are one the features of Open edX that can make a series of questions that have been prepared only accessible and completed within a specific time.

Activate Timed Exams

  1. Ask to IT team to Enable Special Exams before timed exams can be used.

  2. Open your studio > select a course > select Settings > select Advanced Settings > then search for Enable Timed Exams.

  3. Change the value from false to true, because the default setting is false.

Add Timed Exams to Questions

  1. On the Course Outline in studio, select a subsection that has a unit of question or assessment.

  2. Select the Configure icon for subsection.

  3. Click Advanced and select Timed under Set as a Special Exam.

  4. Set the duration then click Save to save the changes.

    Apart from the duration that appears, you can use a certain duration as you wish, just manually type in the hours and minutes.

Taking a Timed Exam as Student

  1. Go to the main page of the course on the LMS.

  2. Open the unit for which the timed exams feature has been set.

  3. Click I am ready to start this timed exam to open the questions and the time will also start the countdown.

  4. You can answer the questions and if the time is up, the question page will be automatically reloaded. Make sure you hit the Submit button before the time runs out so that your answer can be saved to databases.

Resetting a Timed Exam for Specific Student

  1. Go to the Instructors page.

  2. Click Special exams.

  3. Click Student Special Exam Attempts, a list of students who have accessed units that have a timed exam feature.

  4. Click the X icon to clear the student time data.

  5. Until this step, you have reset timed exams for specific students.

Giving Extra Time for Specific Student

  1. Go to the Instructors page.

  2. Click Special exams.

  3. In the allowances section, click Add Allowance.

  4. Enter the additional minutes you want to give, also enter the username you want to give additional time, then click Save to save changes.

  5. Until this step, you have given additional time to specific students.
