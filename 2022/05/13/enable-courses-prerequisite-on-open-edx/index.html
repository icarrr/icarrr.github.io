Enable Courses Prerequisite on Open edX

Course teams can set prerequisites for a courses. Learners must complete the prerequisite courses before open the a specific courses.

To enable this features, follow the steps below because this is not enable by default:

  1. Ask to IT team to Enable Prerequisite before prerequisite course can be used.

  2. Also ask to IT team to enable courses prerequisite. Open and edit LMS configuration (/edx/app/edxapp/lms.env.json or /edx/etc/lms.yml and /edx/app/edxapp/cms.env.json or /edx/etc/studio.yml).

    Then find the "FEATURES" configuration, add ENABLE_PREREQUISITE_COURSES into "FEATURES" array and set to true. See the following codes snippet:

    ...
FEATURES:
    ENABLE_PREREQUISITE_COURSES: true
...

    Restart LMS and CMS services.

  3. Open your studio > select a course > select Settings > select Schedule & Details, then scroll down to the Requirements section.

  4. Select the course from the drop down menu that you would like to make the prerequisite course.

  5. Until this step, courses prerequisite has been set up.
