Check Open edX Version Installed

You can check it from source code of frontend or site of your LMS, it should include the following line:

<meta name="openedx-release-line" content="lilac" />

See for instance the demo platform: https://demo.openedx.overhang.io/ (Ctrl+U to view the source code)
