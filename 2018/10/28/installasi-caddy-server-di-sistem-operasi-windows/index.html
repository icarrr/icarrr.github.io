Installasi Caddy Server di Sistem Operasi Windows

Setelah berkenalan dengan si Caddy Server pada beberapa hari yang lalu, sekarang waktunya bermain dengan si Caddy Server.

Apa Saja yang dibutuhkan? Pastikan anda telah memiliki file master Caddy Server, jika belum silahkan dari situs resminya.

Menjalankan Caddy Server pada sistem operasi windows

  1. Buka file hasil download Caddy Server

  1. Copy semua isi dari file master tersebut

  1. Tentukan path Caddy, buat folder baru yang akan menampung semua file yang telah dicopy kemudian paste di folder tersebut

  1. Untuk menjalankan Caddy Server pada sistem operasi windows cukup klik 2 kali pada file caddy.exe

  1. Jika ada window baru terbuka seperti ini, pilih allow access, selain itu ada window command prompt yang terbuka yang memberikan informasi port untuk mengakses web server anda.

  1. Buka web browser anda dan masukkan url localhost:port

  1. Jika muncul tulisan 404 not found, berarti Caddy berhasil dijalankan di sistem operasi windows anda. Web browser menampilkan tulisan tersebut karena direktori root belum ditentukan.

Mudahkan caranya, untuk penentuan direktori rootnya akan di tulis pada postingan berikutnya.
