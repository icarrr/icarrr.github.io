Python Tidak Memiliki Objek SSL_ST_INIT

Jika akan menghubungkan perangkat anda dengan perangkat lain menggunakan ansible namun ketemu error ERROR! Unexpected Exception: ‘module’ object has no attribute ‘SSL_ST_INIT’, sebenarnya itu hanya membutuhkan paket pyOpenSSL yang dapat diinstall dengan command pip install pyOpenSSL
