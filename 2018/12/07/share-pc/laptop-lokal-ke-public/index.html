Ngrok

Ngrok merupakan salah satu software yang dapat membuat sebuah tunnel atau terowongan yang aman melalui jalur internet agar dapat mengakses pc/laptop lokal tanpa harus memiliki ip publik sendiri.

Jika masih ingin berkenalan lebih lanjut, bisa di stalking ke official sitenya :D https://ngrok.com

Langkah - langkah menggunakan ngrok:

1) Buka shell linux dan login sebagai super user, kemudian download ngrok menggunakan command

wget https://bin.equinox.io/c/4VmDzA7iaHb/ngrok-stable-linux-amd64.zi

2) Ekstrak file ngrok yang telah didownload dengan command

unzip ngrok-***zip

3) Hasil dari ekstrak file .zip yaitu sebuah file yang akan menjadi sebuah command untuk mengaktifkan tunnel, berikan hak akses terlebih dahulu ke file tersebut dengan command

chmod +x ngrok

4) Pindahkan file ngrok ke direktori command linux agar memudahkan membuat tunnelnya dilain hari

mv ngrok /usr/local/bin

5) Sekarang tunnel sudah dapat diaktifkan dengan command

ngrok http 80
